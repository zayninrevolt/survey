Heating Survey Form

Fill in the form and the spreadsheet-style sheets update automatically. Use Save PDFs to create one PDF for Front/Rads and one PDF for the H&S pages.

This is a static HTML file, so it can run locally without a server. It does not send any data anywhere.

Save PDFs

This will save them one at a time so the browser does not block the second page. Front/Rads is set to landscape; H&S is set to portrait.

In the save/print window, enable Background graphics if colours do not appear.
